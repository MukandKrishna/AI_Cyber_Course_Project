# EDA Figures

This folder contains the main exploratory plots used to inspect the PASYDA feature tables.

## Files

- `01_dataset_overview.png` - high-level view of the dataset structure and label balance.
- `02_positive_vs_negative_boxplots.png` - comparison of positive and negative pairs across key numeric features.
- `03_positive_pair_ranks.png` - rank-based view of how positive pairs compare within each scenario.
